# TASK MANAGEMENT APP
 
Abstract: Task Management Application- lists the day to day work like a record. We can add any numbers of tasks that need to be done for the day. Updating of task can
also be done for the added task. At the end of the day, after completion of the task we can also delete task from the list of tasks. The changes made will be reflected
in the database. Simply Task Management Application is the list of records of the day tasks in which a task can be added, updated during the task performance and deleted after completion of the day work.

Run: npm run dev
